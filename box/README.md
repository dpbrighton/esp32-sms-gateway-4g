# SMS Gateway Enclosure

3D printed enclosure for the ESP32 SMS Gateway.

## Files
- `sms_gateway_box.stl` — Main box body
- `sms_gateway_lid.stl` — Friction fit lid (3 sided lip, open back)

## Dimensions
- External: 94 x 81 x 71mm (box body)
- Internal: 88 x 75 x 68mm
- Print bed required: 100 x 100mm minimum

## Print Settings
- Material: PLA
- Layer height: 0.2mm
- Infill: 20%
- Walls: 3 perimeters
- No supports required

## Holes
- Front wall: USB slot 40x22mm, Ethernet 20x22mm
- Right wall: 9mm flash button hole
- Back wall: 10mm SMA antenna hole
